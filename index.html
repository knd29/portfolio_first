制作の流れ

  1. HTML
  2. CSS

  1. HTML

    機械が判別できるように文章の構造を定義します。検索結果の表示順位に影響し、 より見やすく、より有益と判断されたサイトから順に表示されます。

    <h1>ここはタイトルです</h1>

    <p>ここは段落です</p>

    <img ここに画像を表示します>

  2. CSS

    HTMLで定義した文章をスタイリングします。今度は機械ではなく、人が見やすいように レイアウトを変えたり、装飾を加えたりします。

    今日のランチ

    今日は牛肉・にんじん・玉ねぎ・ジャガイモを炒め、水を加えて煮込み、 複数のスパイスや調味料をブレンドしたブロック状のものを加えてよく溶かし、 とろみがつくまでさらに煮込んだものを、白米と一緒に食べてみました！

  3. JavaScript

    ページに動きをつけます。ハンバーガーメニューと呼ばれる、三本線のメニュー アイコンはJavaScriptによって制御されています。先ほどの｢装飾する｣｢レイアウト 変更｣のボタンもコントロールしています。

  4. WordPress

    コンテンツマネジメントシステム(CMS: Contents Management System)を使うこと で、プログラミングしなくてもウェブサイトを管理できます。 ブログサイトも簡単に 運用できます。現在、全ウェブサイトの約43%は、WordPressで運用されています。

    ”健康食”としてのカレー

    カレーには何種類ものスパイスがブレンドされています。その中には、 元々は漢方薬として日本に伝来していたものも多く、高い健康効果を 発揮することが知られています。

    たとえば、ターメリック(ウコン)に含まれる クルクミンは、一般的に肝機能を高めること でよく知られていますが、他にも美肌や脳機能を高める効果もあります。

    カレーには他にもたくさんのスパイスが含 まれており、素晴らしい作用がまだまだ たくさんあります！

    さあ、カレーを食べましょう！！！
